# Study Time Tracker

Study Time Tracker is a simple Python program that helps you estimate how many hours you will study in a week based on how much you studied today. It compares your estimated weekly study time to a weekly goal and shows your progress as a percentage. This project demonstrates basic Python input, calculations, error handling, and a professional Git/GitHub workflow.

## How to Run

1. Make sure you have Python 3 installed.
2. Open a terminal in the project folder (where `app.py` is located).
3. Run one of the following commands:

```bash
python3 app.py
